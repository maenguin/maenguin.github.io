# 객체지향 쿼리 언어 (JPQL)

## 페치 조인 (fetch join)
다대일 관계인 주문과 회원 엔티티가 있다고 하자.  
주문을 가져오면서 회원 정보도 같이 가져오고 싶을땐 어떻게 해야될까?  
  
1. EAGER와 find() 사용  
  fetch 전략을 EAGER로 한 상태에서 em.find()를 사용하면 단일  
  ```java
  em.find(Order.class,1L);
  ```  
  ```sql
  select m.*, 
         o.* 
  from   orders o 
         left join member m 
                on o.member_id = m.member_id 
  where o.order_id = 1
  ```
